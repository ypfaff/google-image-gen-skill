== What This Skill Does

This skill enables Claude Code to generate images using Google's Gemini API. When you ask Claude to create, generate, or edit images, it will use this skill automatically.

Features:

* Generate images from text prompts
* Edit existing images with AI
* Use reference images for style consistency
* Support for multiple aspect ratios
* Batch generation of multiple variations

== Requirements

* https://docs.astral.sh/uv/[uv] package manager
* Google AI API key from https://aistudio.google.com/apikey[Google AI Studio]

== Installation

=== 1. Copy the Skill

Copy the `.claude/skills/google-image-gen/` folder into your project:

[source,bash]
----
# From this repo
cp -r .claude/skills/google-image-gen /path/to/your/project/.claude/skills/
----

=== 2. Create .env File

[source,bash]
----
cd /path/to/your/project/.claude/skills/google-image-gen
cp .env.example .env
----

Edit `.env` and add your API key:

[source]
----
GOOGLE_AI_API_KEY=your_actual_api_key_here
----

WARNING: Make sure `.env` is listed in your project's `.gitignore` to avoid accidentally committing your API key.

=== 3. Done

Claude Code will automatically detect and use the skill when you ask it to generate images.

== Usage Examples

Just ask Claude Code naturally. Here are some example prompts:

=== Simple Image Generation

[source]
----
Generate an image of a red sports car on a mountain road and save it as car.png
----

[source]
----
Create a minimalist logo for a coffee shop, save as logo.png with 1:1 aspect ratio
----

=== With Specific Aspect Ratios

[source]
----
Generate a YouTube thumbnail showing a coding tutorial, use 16:9 aspect ratio
----

[source]
----
Create a phone wallpaper with abstract geometric shapes, aspect ratio 9:16
----

=== Editing Existing Images

[source]
----
Edit the image input.png and change the background to a sunset sky
----

[source]
----
Take product.png and remove the background, make it transparent
----

=== Using Reference Images

[source]
----
Generate an image in the same style as reference.png but with a cat instead
----

[source]
----
Create a new image matching the color palette of style.png, showing a mountain landscape
----

=== Multiple Variations

[source]
----
Generate 3 variations of app icons: a rocket, a star, and a lightning bolt
----

=== With Style Templates

You can create custom style templates as markdown files with a `## Prompt Template` section:

.my-style.md
[source,markdown]
----
## Prompt Template

```
Minimalist flat design illustration of {subject}.
Clean lines, bold colors, white background.
Modern corporate style, suitable for presentations.
```
----

Then ask Claude:

[source]
----
Generate an image of a laptop using the style template my-style.md
----

== Supported Aspect Ratios

[cols="1,2"]
|===
|Ratio |Use Case

|`1:1`
|Social media posts, profile pictures

|`3:4`
|Portrait photos

|`4:3`
|Standard landscape

|`4:5`
|Instagram portrait

|`5:4`
|Large format prints

|`9:16`
|Phone wallpapers, Stories, TikTok

|`16:9`
|YouTube thumbnails, presentations (default)

|`21:9`
|Ultrawide displays, cinematic
|===

== Troubleshooting

[cols="1,2,2"]
|===
|Error |Cause |Solution

|`GOOGLE_AI_API_KEY not found`
|Missing or invalid .env file
|Create `.env` with your API key in the skill directory

|`Rate limit exceeded`
|Too many API requests
|Wait a few minutes or upgrade to paid tier

|`uv: command not found`
|uv not installed
|Install via `curl -LsSf https://astral.sh/uv/install.sh \| sh`
|===
